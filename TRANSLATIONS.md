## Beitragen von Übersetzungen

Wenn Sie eine neue Sprache hinzufügen möchten, folgen Sie bitte diesem Schema:

### Benennung
- Hauptdokumentation: `README.{lang}.md` (z.B. `README.fr.md` für Französisch)
- OCR-Scanner Anleitung: `OCR-SCANNER.{lang}.md` (z.B. `OCR-SCANNER.fr.md` für Französisch)

### Sprach-Codes
Verwenden Sie ISO 639-1 Sprach-Codes:
- `de` - Deutsch
- `en` - Englisch
- `fr` - Französisch
- `es` - Spanisch
- `it` - Italienisch
- `ja` - Japanisch
- `zh` - Chinesisch
- `pt` - Portugiesisch
- `ru` - Russisch

### Anforderungen für neue Übersetzungen

1. **Korrektheit**: Achten Sie auf genaue technische Übersetzung
2. **Konsistenz**: Verwenden Sie einheitliche Terminologie
3. **Aktualität**: Stellen Sie sicher, dass die Übersetzung mit der aktuellen Version übereinstimmt
4. **Links**: Aktualisieren Sie Links zu Übersetzungen in:
   - `README.md`
   - `TRANSLATIONS.md`

### Checkliste für neue Übersetzungen

- [ ] `README.{lang}.md` erstellt
- [ ] `OCR-SCANNER.{lang}.md` erstellt (optional)
- [ ] Links in englischer `README.md` hinzugefügt
- [ ] Diese Datei (`TRANSLATIONS.md`) aktualisiert
- [ ] Alle Dateien formatiert und validiert
- [ ] Pull Request erstellt
